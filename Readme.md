# Ondora 🌊

> **"Where rhythm meets geometry."**

Ondora was born from the moments when I close my eyes while listening to a track and see an entire world taking shape. I wanted to build a bridge between those internal visuals, the shifting waves, the glowing ambients, and the rhythmic pulses and the screen. This is a tool to visualize exactly what the mind sees when the music takes over.

## 🧬 The Anatomy of Ondora

The name reflects the fusion of movement and atmosphere:

- **Onda (Wave):** Representing the raw, physical energy of the audio frequencies; the spikes, the rhythm, and the motion.
- **Aura (Smooth):** Representing the ambient glow, the transitions, and the immersive environment that surrounds the sound.

Together, they form **Ondora**: A smooth, wave-driven sensory experience for music.

---

## The Stack

- **Runtime:** Electron (Desktop performance & File access)
- **Frontend:** React + Tailwind CSS (Custom, sleek utility-first UI)
- **Graphics:** Three.js / WebGL (For the 3D mental landscapes)
- **Audio Engine:** Web Audio API (FFT Analysis for real-time response)

## Features

- **Mental Waveforms:** High-fidelity vertex displacement that mirrors the "visuals" of high notes and deep bass.
- **Atmospheric Glow:** Real-time lighting and "aura" effects that change intensity based on track energy.
- **Interactive Exploration:** Use your mouse to navigate through the audio-generated world.
- **Tailwind Precision:** No bloated libraries—just custom, responsive controls.

### Prerequisites

- Node.js (v21+)
- npm

### Installation

1. **Clone the repo:**
   ```bash
   git clone https://github.com/ndizeyedavid/ondora
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the app:**
   ```bash
   npm run dev
   ```
